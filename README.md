# Starter kit

Stacks: Docker, php, mariadb, nginx, Symfony

## Start the project

1. Edit global variable .env (refer to .env.dist)

2. Use "Make" commands.

`make up`

## Tests

Run `make phpunit`

## Coding standards

### PHPStan:

Run `make stan`

### php-cs-fixer:

Run `make cs-fixer`
